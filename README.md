## Usage
For all characters:
```ino
#include "CustomLCDchars.h"
```

Seperately:
```ino
#include "CustomLCDcharsEmoji.h"
#include "CustomLCDcharsGeo.h"
#include "CustomLCDcharsUtil.h"
```
